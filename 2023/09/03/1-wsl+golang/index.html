WSL中配置golang环境

Posted at 2023-09-03 Linux golang 

配置WSL

参考文章：WSL安装

切换root用户

sudo passwd root
su

限制Vmmem内存

  1. 按下Windows + R键，输入%UserProfile%并运行进入用户文件夹

  2. 新建文件.wslconfig，然后记事本编辑

  3. 填入以下内容并保存, memory为系统内存上限，这里我限制最大2GB，可根据自身电脑配置设置

[wsl2]
memory=4GB
swap=8GB
localhostForwarding=true
  1. 输入wsl --shutdown来关闭当前的子系统，重启wsl

自动安装

sudo apt install golang-go
apt remove golang-go

用这样的办法自动安装的golang并非最新版

手动安装

  1. 查看google发布的golang版本，选择合适的版本安装。
wget https://studygolang.com/dl/golang/go1.15.2.linux-amd64.tar.gz
  1. 解压到/usr/local
tar -C /usr/local -xzf go1.15.2.linux-amd64.tar.gz
  1. 设置环境变量
vim /etc/profile

export GOPATH=/home/xiaocai/go
export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin

source /etc/profile
  1. 其他设置
# 配置代理
go env  -w GOPROXY=https://goproxy.cn,https://goproxy.io,direct

# Go Moudle
go env -w GO111MODULE=on
  1. 查看环境是否安装成功
go env
go version

golang配置

  • 添加语言运行时

  • 使用whereis go，得到go: /usr/local/go /usr/local/go/bin/go /mnt/d/Go/bin/go.exe

    • 第一个参数是go sdk文件夹

    • 第二个参数/usr/local/go/bin/go对应Go可执行文件参数

    • 第三个是Windows10里面的go环境映射到了wsl2
